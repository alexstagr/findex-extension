# Findex Extension

Perform search using custom search engines directly from a popup.  
You can use pre-defined search engines or add your own with custom triggers.

---

## Screenshot

![](https://raw.githubusercontent.com/alexanstgr/findex-extension/refs/heads/main/thumbnail.png)

## Features

- Quickly search using multiple search engines.
- Manage user saved localstorage search engines.
- Add your own search engines with a custom `@trigger`.
- Auto-updating icons (favicons) for each engine.
- Simple popup interface with live engine selection.
- Prevents duplicate triggers.

---

## Commands

- #manager (Opens engines manager window )
- #register (Opens the registration window)

## Testing

1. Clone or download this repository.
2. Open your browser (Firefox or Chromium-based) and go to **Extensions / Add-ons**.
3. Enable **Developer Mode** (or Debug Mode in Firefox).
4. Click **Load unpacked** (or "Load Temporary Add-on" in Firefox).
5. Select the folder containing this project.
